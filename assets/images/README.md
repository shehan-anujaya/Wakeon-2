# Image Assets

This directory contains image assets for the WakeOn app.

## Expected Images

### App Icon
- `app_icon.png` - Main app icon (1024x1024 for stores)

### UI Elements
- `face_guide.png` - Face positioning overlay guide
- `logo.png` - App logo for splash/home screen

### Status Icons
- `status_alert.png` - Alert/awake status indicator
- `status_drowsy.png` - Drowsy status indicator  
- `status_critical.png` - Critical/microsleep indicator

## Specifications

| Asset | Size | Format |
|-------|------|--------|
| App Icon | 1024x1024 | PNG |
| Logos | Variable | SVG/PNG |
| Icons | 48x48 - 96x96 | PNG |
| Guides | 300x400 | PNG (transparent) |

## Color Palette

- **Safe Green**: #2ECC71
- **Warning Yellow**: #F1C40F  
- **Critical Red**: #E74C3C
- **Background Dark**: #1A1A2E
